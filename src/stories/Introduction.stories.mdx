# Introducción

En esta librería podras encontrar los componentes utilizados entre los proyecto Global

## Instalación

Para instalar la librería en tu proyecto

`yarn add @lissettebm/ui-example`

## Uso

Para usar los componentes primero que todo debes importar el siguiente archivo en la raíz de tu proyecto:

`import "@lissettebm/ui-example/dist/scss/styles.scss"`

Luego puedes navegar entre los distintos componentes en el menú lateral.<br/>
En cada uno de ellos encontrarás la documentación necesaria para utilizarlos.
